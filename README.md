# Deloitte Forward Deployed Engineer candidate vision

Independent candidate campaign by Russell Dudek for Deloitte's Forward Deployed Engineer, Frontier GenAI role.

The public experience argues that frontier-model capability creates enterprise value only when it becomes an owned, observable, governed client workflow. It includes the Enterprise AI Deployment Assembly, an interactive deployment-posture lab, a two-page role-aligned résumé, a one-page cover letter, an interview thesis brief, a 90-day entry plan, and a Forward Deployment Contract.

Open `index.html` for the candidate vision. Every document route includes a direct PDF download; no print-dialog controls are used.
